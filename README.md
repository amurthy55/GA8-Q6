# Employee Compensation Beeswarm Visualization

This repository contains a RAWGraphs beeswarm plot that visualizes
salary distributions across different departments and experience levels.
It is designed for executive decision-making, pay equity analysis, and
human capital strategy.

## Dataset
The dataset consists of 20 synthetic records with:
- department (categorical)
- level (experience level)
- salary (continuous)

The data simulates a realistic compensation model used in enterprise HR analytics.

## Visualization Method
Created using RAWGraphs (https://rawgraphs.io)

- Chart Type: Beeswarm Plot
- X-axis: Salary
- Groups (Y): Level (Junior → Executive)
- Colors: Department
- Exported as PNG (450×450)
